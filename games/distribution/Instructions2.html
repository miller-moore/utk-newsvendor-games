Here are 2 examples of calculating the profit of a round:

  1. Example 1:
    • selling price = {{ 20 | cu }} / unit
    • purchasing price = {{ 7.5 | cu }} / unit
    • salvage value = {{ 5 | cu }} / unit
    • your order quantity = 50 units
    • demand = 20 units

    Profit = {{ 20 | cu }} * 20 units [units sold = Minimum of demand and order quantity]
      - {{ 7.5 | cu }} * 50 units [order quantity]
      + {{ 5 | cu }} * 30 units [salvage inventory= order quantity - units sold]
      = {{ 175 | cu }}
  2. Example 2:
    • Prices of Example 1 apply here.
    • your order quantity = 50 units
    • demand = 100 units

    Profit = {{ 20 | cu }} * 50 units [units sold = Minimum of demand and order quantity]
      - {{ 7.5 | cu }} * 50 units [order quantity]
      + {{ 5 | cu }} * 0 units [final inventory= 0]
      = {{ 625 | cu }}
